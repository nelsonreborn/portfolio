🚀 Projects

🌐 Ardent Universe – Job Board Overhaul + SEO Boost

Problem: Ardent Universe was listing jobs as blog posts, which hurt categorization, discoverability, and SEO scalability.

My Solution:

  • Created a Custom Post Type (Jobs) using CPT UI
  • Migrated existing blog post jobs into the new CPT
  • Added custom fields (Job Type, Experience, Company, Field) using ACF
  • Applied structured data/schema markup for Google Jobs indexing
  • Built a custom homepage template (`page-latest.php`) to show latest jobs dynamically
  • Set up Job Manager plugin to allow companies to submit jobs for admin approval
  • Used Yoast SEO for meta tags, and conducted keyword research with Ahrefs and Keyword Surfer

Impact: Improved job discoverability on Google and streamlined the site's structure for future scalability.

📈 MyJobMag – Blog SEO Strategy
